# JG. SUFFU — CA Intermediate Tracker

A highly responsive, perfectly isolated, single-page web app built to organize and track preparation specifically for the **CA Intermediate Examinations (May/November 2026)**.

## Key Features
- **Glassmorphism UI**: Beautiful, premium design with dark/light mode support.
- **Dynamic Planner**: Day-by-day tracking for all 6 subjects.
- **Test Logging**: Record scores, coverage, and confidence levels.
- **Schedule Management**: Customizable timetables for different day types.
- **PWA Ready**: Install as a mobile app for quick access.
- **Firebase Sync**: Real-time cloud storage and authentication.

## Transformation Note
This app was successfully refactored from the CA Foundation Tracker to the CA Intermediate Tracker with updated subject matter, reconfigured backend, and a complete UI overhaul.
